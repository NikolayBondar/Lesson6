find_ex.sh - скрипт проверяет файлы в папке /usr/bin и находит те, у которых есть атрибут на запуск и чтение. 
Список файлов выводит в файл usr_executables.txt

read_file.sh - скрипт использует текстовый файл  usr_executables.txt и выводит количество строк. 
Затем у первых 10 результатах и обрезает путь к файлу по символу “/”, 
оставляя список только имен файлов. Затем перенаправляет их в новый файл > cmd_names.

find_py.sh - скрипт находит все файлы *.py  и записывает поток в  file  py_scripts, а ошибки в py_errors.

descr.sh - скрипт который проверяет дескриптор открытых файлов и  процессов пользователя  и выводит их в файл вместе с PID процесса 
в формате PID, USER, FD, NAME.
